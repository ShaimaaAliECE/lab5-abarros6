Project Summary

Append the extensions (the projects' endpoints) "/functionality1", "/:functionality2" and "/functionality3" as suffixes to localhost:80
in the browser address bar to view respective pages or navigate to them by interacting with the GUI links below.

Try pasting the following routes into the URL to quickly test the three functionalities.

Functionality 1: localhost:80/functionality1

Functionality 2: localhost:80/.net

Functionality 3: localhost:80/functionality3?city=Amsterdam

Link to github repository

Functionalities

Functionality 1: Return the categories mentioned in the jobs.json file and how many times each category is mentioned.

Functionality 2: Return all the jobs with a given category sent as a parameter

Functionality 3: Return all the jobs in a given city sent as a querystring
